# Dashboard Arsiparis

Dashboard ini menyajikan informasi komprehensif mengenai Jabatan Fungsional Arsiparis, mencakup profil, kinerja, serta hasil monitoring dan evaluasi tahun 2025.

Data yang ditampilkan memberikan gambaran mengenai:
- Sebaran dan karakteristik arsiparis  
- Kinerja dan capaian tugas jabatan  
- Hasil monitoring dan evaluasi  
- Pengembangan profesi dan kompetensi  

Seluruh informasi diolah dari data yang dihimpun secara nasional, termasuk hasil survei daring pada Desember 2025, untuk menghasilkan gambaran yang objektif dan menyeluruh.

## Tujuan

Dashboard ini bertujuan untuk:
- Menyediakan informasi kearsipan secara terintegrasi  
- Mendukung pengambilan kebijakan berbasis data  
- Mengidentifikasi potensi dan tantangan dalam pengelolaan kearsipan  
- Memperkuat pembinaan dan pengembangan karier arsiparis  

## Dasar Hukum

Pengelolaan dan penyajian data mengacu pada:
- Undang-Undang Nomor 43 Tahun 2009 tentang Kearsipan  
- Peraturan Pemerintah (PP) Nomor 28 Tahun 2012 tentang Pelaksanaan Undang Undang Nomor 43 Tahun 2009 Tentang Kearsipan
- Peraturan Menteri Pendayagunaan Aparatur Negara dan Reformasi Birokrasi Nomor 48 Tahun 2014 tentang Jabatan Fungsional Arsiparis

## Catatan

Data yang disajikan merupakan hasil olahan dari berbagai sumber dan mencerminkan kondisi pada periode pengumpulan data.
